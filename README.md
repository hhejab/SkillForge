# SkillForge

SkillForge is a web-based system for managing training courses and certifications.

The idea is simple: coordinators create courses and schedule sessions, trainees can enroll and track their progress, and instructors record results. Once a trainee finishes the required courses, they can receive a certificate.

---

## Features

- Different user roles (Trainee, Instructor, Coordinator)
- Create and manage courses
- Schedule sessions (time, room, instructor)
- Enroll in courses
- Track progress
- Pass / Fail results
- Certificate generation
- Basic reporting
- API for data and certificate verification


---

## How to run

1. Clone the repo
2. Open the solution in Visual Studio 2022
3. Set SkillForge.MVC as startup project
4. Run the project
